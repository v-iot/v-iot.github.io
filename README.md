:--------: | :------------: |

[viot-camera(摄像机设置和预览)](https://github.com/v-iot/viot-camera) | 常用相机的设置和预览方式 |

[viot-3d(三维坐标定位)](https://github.com/v-iot/viot-3d) | 双目相机下标签（物体）的三维坐标定位 |
[viot-utils(工具包)](https://github.com/v-iot/viot-utils) | 其它VI项目需要的公共工具类 |
